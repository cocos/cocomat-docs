# cocomat 简介

`Cocos Creator` 凭借其跨平台的特性以及简单易用的编辑器工具，成为了当下最受欢迎的游戏开发引擎以及在线教育应用开发引擎之一。越来越多的公司和团队选择使用 `Cocos Creator` 进行开发。

因此，我们希望能开发一套可在多个业务间复用的高性能、高效能 `Cocos Creator` 公共组件框架，旨在助力 `Cocos Creator` 开发者提升开发效率，更专注于业务开发，即 `cocomat`。

`cocomat` 目前主要包括：

* 一系列 UI 组件，如 `Toast`、`Loading`、`BackBtn` 等；
* 工具组件，如场景管理、截图工具、音频播放、View 管理、内存管理、视力保护等；
* 一套自研无层级视频播放器。
